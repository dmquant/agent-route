Key Design Decisions

Anthropic Managed Agent Principles Applied

The architecture follows the core tenets from Anthropic's "Building effective agents" research:

Brains are cattle, not pets

If the orchestrator crashes, wake(sessionId) replays the event log. No in-memory state is irreplaceable.

Hands call tools, Brains call hands

The orchestrator never knows if it's calling a CLI process or an HTTP API — it only knows the Hand interface.

Context lives outside the LLM

All context is durably stored in the event log. The ContextEngine decides what to inject per turn.

Harnesses encode stale assumptions

Per-agent configs (context limits, compaction, retry) are explicit and hot-swappable without session disruption.
